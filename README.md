# Backtest de moyennes mobiles en Python

Ce projet montre comment backtester une stratégie simple
basée sur les moyennes mobiles (SMA / EMA) en Python.

## Contenu
- Téléchargement de données via Yahoo Finance
- Calcul des SMA et EMA
- Backtest sans look-ahead bias
- Comparaison avec le Buy & Hold
